## 概述

欢迎访问 chengenbao.github.io（以下简称"本网站"）。本隐私政策说明本网站如何收集、使用和保护您的个人信息。

## 信息收集

### 访问日志

本网站通过 Google Analytics 收集匿名访问数据，包括：

- 页面浏览量
- 访问来源
- 浏览器类型和操作系统
- 大致地理位置（国家/城市级别）

这些数据用于了解网站访问情况，不会识别您的个人身份。

### 评论系统

本网站使用 Giscus 评论系统（基于 GitHub Discussions）。如您发表评论，需要使用 GitHub 账号登录，相关数据受 [GitHub 隐私政策](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) 约束。

### 广告

本网站可能使用 Google AdSense 展示广告。Google 可能使用 Cookie 根据您对本网站及其他网站的访问情况向您展示相关广告。您可以通过 [Google 广告设置](https://www.google.com/settings/ads) 选择退出个性化广告。

## Cookie

本网站使用以下 Cookie：

- **Google Analytics**：用于统计访问数据（匿名）
- **Google AdSense**：用于广告展示和频率控制
- **Giscus**：用于保持评论系统的登录状态

您可以通过浏览器设置管理或禁用 Cookie，但这可能影响网站部分功能的正常使用。

## 第三方服务

本网站使用以下第三方服务，请参阅其各自的隐私政策：

- [Google AdSense](https://policies.google.com/privacy)

本网站不主动收集或存储您的个人身份信息。网站托管于 GitHub Pages，数据安全受 GitHub 保障。
