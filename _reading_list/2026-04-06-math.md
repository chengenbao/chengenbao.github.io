## 【四/五年级】青蛙的颜色变化

**难度：** ⭐⭐⭐（AMC 8 Problem 21，竞赛级比例方程）

**题目来源：** [2024 AMC 8 Problem 21](https://artofproblemsolving.com/wiki/index.php/2024_AMC_8_Problems/Problem_21)

**题目：**

一群青蛙住在一棵树上。青蛙在阴凉处会变成**绿色**，在阳光下会变成**黄色**。

最初，绿色青蛙和黄色青蛙的比例是 $3:1$。

之后，**3只绿色青蛙**移到了阳光下，**5只黄色青蛙**移到了阴凉处。

现在，绿色青蛙和黄色青蛙的比例变成了 $4:1$。

**请问现在绿色青蛙和黄色青蛙的数量差是多少？**

（A）10　　（B）12　　（C）16　　（D）20　　（E）**24** ✓

---

### 解题过程

**关键理解：** 青蛙的颜色由**位置决定**（阴凉=绿，阳光=黄），移动会改变颜色！

**设未知数：**

设最初绿色（阴凉）青蛙有 $3x$ 只，黄色（阳光）青蛙有 $x$ 只。

**分析移动：**
- 3只绿色 → 移到阳光（变黄）：绿色**减少** 3，黄色**增加** 3
- 5只黄色 → 移到阴凉（变绿）：黄色**减少** 5，绿色**增加** 5

所以移动后：
$$\text{绿色} = 3x - 3 + 5 = 3x + 2$$
$$\text{黄色} = x + 3 - 5 = x - 2$$

**列方程（新比例为 $4:1$）：**

$$\frac{3x + 2}{x - 2} = 4$$

$$3x + 2 = 4(x - 2)$$

$$3x + 2 = 4x - 8$$

$$x = 10$$

**求差值：**

- 移动后绿色：$3 \times 10 + 2 = 32$ 只
- 移动后黄色：$10 - 2 = 8$ 只
- 差值：$32 - 8 = \boxed{24}$

**答案：(E) 24**

---

### 知识背景

这道题考察**比例与一次方程**，是竞赛中常见的"变化量分析"题型。核心技巧是先弄清楚**每次移动对各组数量的影响**，再设未知量建立方程。

注意一个容易犯的错误：题目说的是3只绿色青蛙和5只黄色青蛙互换位置，**不是直接变色**——青蛙的颜色由位置决定，所以移动之后双向都发生了变化（绿色净增2，黄色净减2）。

类似题型：一袋红球和白球，交换几颗后比例改变，求原来数量。

## 【四/五年级】网格上的线段

**难度：** ⭐⭐⭐⭐（AMC 8 Problem 23，数论 GCD 应用）

**题目来源：** [2024 AMC 8 Problem 23](https://artofproblemsolving.com/wiki/index.php/2024_AMC_8_Problems/Problem_23)

**题目：**

Rodrigo 在一张很大的方格纸上作图。

首先，他画了从点 $(0, 4)$ 到 $(2, 0)$ 的线段，并涂色了**所有内部与线段相交的格子**，共涂了 **4 个**格子。

接下来，他画了从点 $(2000, 3000)$ 到 $(5000, 8000)$ 的线段，同样涂色内部与线段相交的格子。

**他这次会涂多少个格子？**

（A）6000　　（B）6500　　（C）**7000** ✓　　（D）7500　　（E）8000

---

### 解题过程

**第一步：从小例子找规律**

验证第一条线段 $(0,4)$ → $(2,0)$：
- $\Delta x = 2,\quad \Delta y = 4,\quad \gcd(2, 4) = 2$
- 穿越格子数 $= \Delta x + \Delta y - \gcd(\Delta x, \Delta y) = 2 + 4 - 2 = 4$ ✓（与题目吻合！）

**一般公式（Crossing Number Formula）：**

> 线段从 $(x_1, y_1)$ 到 $(x_2, y_2)$，它穿过的格子数为：
> $$N = |\Delta x| + |\Delta y| - \gcd(|\Delta x|, |\Delta y|)$$

**第二步：计算第二条线段**

$$\Delta x = 5000 - 2000 = 3000$$
$$\Delta y = 8000 - 3000 = 5000$$

**计算 $\gcd(3000, 5000)$（辗转相除法）：**

$$5000 = 1 \times 3000 + 2000$$
$$3000 = 1 \times 2000 + 1000$$
$$2000 = 2 \times 1000 + 0$$

所以 $\gcd(3000, 5000) = 1000$。

**代入公式：**

$$N = 3000 + 5000 - 1000 = \boxed{7000}$$

**答案：(C) 7000**

---

### 知识背景

这道题背后是数论中的**格点与整数线段**经典定理：

线段从一个整数坐标点到另一个整数坐标点时，会穿过若干个单位格子。公式 $\Delta x + \Delta y - \gcd(\Delta x, \Delta y)$ 中：
- $\Delta x$ 是水平穿越的边界数
- $\Delta y$ 是竖直穿越的边界数  
- $\gcd(\Delta x, \Delta y)$ 是恰好经过格点（正好走在格子角上）的次数，这些点**不额外增加**穿越数

这个公式也叫 **Pick 定理的推论**，在竞赛中频繁出现。辗转相除法（欧几里得算法）是计算 GCD 的标准方法，建议熟练掌握！
