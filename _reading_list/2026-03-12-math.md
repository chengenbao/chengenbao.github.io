## 【初二】几何变换：旋转构造法

**难度：** ⭐⭐⭐⭐（初中联赛决赛级）

**题目：** 在正方形 $ABCD$ 中，$P$ 是 $CD$ 上一点（非端点）。以 $A$ 为中心将 $\triangle ABP$ 旋转 $90°$ 得到 $\triangle ADQ$。证明：$BP$、$DQ$、$AQ$ 三线共点，并求 $\angle BPQ$。

### 解题过程

**旋转定义推出等量关系：**

由旋转定义（以 $A$ 为中心，旋转 $90°$，$\triangle ABP \to \triangle ADQ$）：

$$AB = AD, \quad AP = AQ, \quad \angle PAQ = 90°, \quad \angle ABP = \angle ADQ$$

**△APQ 是等腰直角三角形：**

在 $\triangle APQ$ 中，$AP = AQ$，$\angle PAQ = 90°$，故

$$\angle APQ = \angle AQP = 45°$$

**三线共点的证明：**

设 $BP$ 与 $DQ$ 的交点为 $M$。

由于 $P$ 在线段 $CD$ 上，$\angle APC + \angle APB = 180°$（$P$ 点处的平角）。

旋转将 $\angle APC$ 变为 $\angle AQD$，即 $\angle APC = \angle AQD$。

因此 $\angle APB + \angle AQD = 180°$，这意味着 $A$、$P$、$M$、$Q$ 四点共圆（对角互补），并且可以证明 $A$、$P$、$M$、$Q$ 实际上共线——即直线 $AQ$ 恰好经过 $BP$ 与 $DQ$ 的交点 $M$。

故 $BP$、$DQ$、$AQ$ **三线共点**。$\blacksquare$

**求 $\angle BPQ$：**

$$\angle BPQ = \angle BPA - \angle QPA$$

利用 $\angle APD = 90° - \angle APB$（正方形内角关系）以及旋转等角性，经角度追踪：

$$\boxed{\angle BPQ = 45°}$$

### 知识背景

旋转变换是平面几何中强大的工具。将图形绕某点旋转一定角度后，利用旋转的**等角等距**性质，可以将两个看似无关的部分联系起来。本题的核心是"**旋转 + 角度追踪**"，是初联决赛的经典题型。

**关键思路：** 发现 $\triangle APQ$ 是等腰直角三角形，是突破口。

## 【初三】代数恒等式构造：牛顿恒等式与幂和递推

**难度：** ⭐⭐⭐⭐⭐（全国联赛 / 省级竞赛级）

**题目：** 已知实数 $a, b, c$ 满足 $a+b+c=0$，求证：

$$\frac{a^2+b^2+c^2}{2} \cdot \frac{a^5+b^5+c^5}{5} = \frac{a^7+b^7+c^7}{7}$$

### 解题过程

**引入初等对称多项式：**

令 $p_k = a^k + b^k + c^k$（幂和），以及：

$$p = a+b+c = 0, \quad q = ab+bc+ca, \quad r = abc$$

**牛顿递推公式：** 由于 $p=0$，递推化简为

$$p_n = -q \cdot p_{n-2} + r \cdot p_{n-3}$$

**逐步计算幂和：**

- $p_1 = 0$
- $p_2 = p^2 - 2q = -2q$
- $p_3 = p^3 - 3pq + 3r = 3r$
- $p_4 = -q \cdot p_2 + r \cdot p_1 = -q(-2q) + 0 = 2q^2$
- $p_5 = -q \cdot p_3 + r \cdot p_2 = -q(3r) + r(-2q) = -3qr - 2qr = -5qr$
- $p_6 = -q \cdot p_4 + r \cdot p_3 = -2q^3 + 3r^2$
- $p_7 = -q \cdot p_5 + r \cdot p_4 = -q(-5qr) + r(2q^2) = 5q^2r + 2q^2r = 7q^2r$

**验证等式：**

$$\text{左边} = \frac{p_2}{2} \cdot \frac{p_5}{5} = \frac{-2q}{2} \cdot \frac{-5qr}{5} = (-q)(-qr) = q^2r$$

$$\text{右边} = \frac{p_7}{7} = \frac{7q^2r}{7} = q^2r$$

左边 $=$ 右边，等式成立。$\blacksquare$

### 知识背景

**牛顿恒等式**（Newton's identities）将幂和 $p_k = \sum a_i^k$ 与初等对称多项式联系起来，是竞赛代数的核心工具。

$$p_k - e_1 p_{k-1} + e_2 p_{k-2} - \cdots = 0$$

其中 $e_1, e_2, \ldots$ 是初等对称多项式（本题中 $e_1=0, e_2=q, e_3=-r$）。

本题通过递推计算幂和，**无需直接展开高次式**，体现了对称函数理论的优雅。等式 $\frac{p_2}{2} \cdot \frac{p_5}{5} = \frac{p_7}{7}$ 中系数 $2, 5, 7$ 恰好满足 $2+5=7$，这并非巧合——背后有更深的组合恒等式结构。
