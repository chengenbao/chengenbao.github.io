## 【初二】几何变换：旋转构造法

**难度：** ⭐⭐⭐⭐（初中联赛决赛级）

**题目：** 在正方形 $ABCD$ 中，$P$ 是 $CD$ 上一点（非端点）。以 $A$ 为中心将 $\triangle ABP$ 顺时针旋转 $90°$ 得到 $\triangle ADQ$。证明：$BP \perp DQ$ 且 $BP = DQ$。

### 解题过程

**旋转定义推出基本等量关系：**

由旋转定义（以 $A$ 为中心顺时针旋转 $90°$，$\triangle ABP \to \triangle ADQ$）：

$$AB = AD, \quad AP = AQ, \quad \angle BAP = \angle DAQ = 90°, \quad BP = DQ$$

其中 $BP = DQ$ 由旋转保距性直接得到。$\square$

**证明 $BP \perp DQ$：**

设 $\angle BAP = \theta$，则 $\angle DAQ = \theta$（旋转等角）。

注意 $\angle BAD = 90°$（正方形内角），故

$$\angle PAQ = \angle PAB + \angle BAD + \angle DAQ = \theta + 90° + \theta$$

等等，更简洁的方式是直接考虑向量。

设 $A$ 为原点，$B = (1,1)$，$D = (0,0)$（正方形 $ABCD$，$A=(0,1)$），$P = (p, 0)$ 在 $CD$ 上。

顺时针旋转 $90°$ 将 $(x,y)$ 映射到 $(y, -x)$（相对于原点），相对于 $A=(0,1)$ 的变换为：

$$Q = A + R_{-90°}(P - A)$$

其中 $R_{-90°}(x,y) = (y, -x)$，故

$$P - A = (p, -1), \quad R_{-90°}(p,-1) = (-1, -p), \quad Q = (0,1) + (-1,-p) = (-1, 1-p)$$

**向量计算：**

$$\overrightarrow{BP} = P - B = (p-1, -1)$$

$$\overrightarrow{DQ} = Q - D = (-1, 1-p)$$

**验证垂直：**

$$\overrightarrow{BP} \cdot \overrightarrow{DQ} = (p-1)(-1) + (-1)(1-p) = -(p-1) - (1-p) = (1-p) - (1-p) = 0$$

故 $BP \perp DQ$。$\blacksquare$

**验证等长：**

$$\lvert BP \rvert = \sqrt{(p-1)^2 + 1}, \quad \lvert DQ \rvert = \sqrt{1 + (1-p)^2} = \sqrt{(1-p)^2+1}$$

故 $\lvert BP \rvert = \lvert DQ \rvert$。$\blacksquare$

### 知识背景

旋转变换保持距离和角度不变，是平面几何中最强大的工具之一。本题的核心技巧是：**用向量内积验证垂直**。将旋转后的点坐标化，计算向量点积为零，是竞赛中处理"旋转构造+垂直等长"类问题的标准路径。

竞赛中常见的旋转构造题型：在等边三角形、正方形等正多边形上旋转，证明某两条线段垂直且相等，或三线共点。本题结论 $BP \perp DQ$、$BP = DQ$ 正是旋转 $90°$ 的典型特征。


## 【初三】代数恒等式构造：牛顿恒等式与幂和递推

**难度：** ⭐⭐⭐⭐⭐（全国联赛 / 省级竞赛级）

**题目：** 已知实数 $a, b, c$ 满足 $a+b+c=0$，求证：

$$\frac{a^2+b^2+c^2}{2} \cdot \frac{a^5+b^5+c^5}{5} = \frac{a^7+b^7+c^7}{7}$$

### 解题过程

**引入初等对称多项式：**

令 $p_k = a^k + b^k + c^k$（幂和），以及：

$$p = a+b+c = 0, \quad q = ab+bc+ca, \quad r = abc$$

**牛顿递推公式：** 由于 $p=0$，递推化简为

$$p_n = -q \cdot p_{n-2} + r \cdot p_{n-3}$$

**逐步计算幂和：**

- $p_1 = 0$
- $p_2 = p^2 - 2q = -2q$
- $p_3 = p^3 - 3pq + 3r = 3r$
- $p_4 = -q \cdot p_2 + r \cdot p_1 = -q(-2q) + 0 = 2q^2$
- $p_5 = -q \cdot p_3 + r \cdot p_2 = -q(3r) + r(-2q) = -3qr - 2qr = -5qr$
- $p_6 = -q \cdot p_4 + r \cdot p_3 = -2q^3 + 3r^2$
- $p_7 = -q \cdot p_5 + r \cdot p_4 = -q(-5qr) + r(2q^2) = 5q^2r + 2q^2r = 7q^2r$

**验证等式：**

$$\text{左边} = \frac{p_2}{2} \cdot \frac{p_5}{5} = \frac{-2q}{2} \cdot \frac{-5qr}{5} = (-q)(-qr) = q^2r$$

$$\text{右边} = \frac{p_7}{7} = \frac{7q^2r}{7} = q^2r$$

左边 $=$ 右边，等式成立。$\blacksquare$

### 知识背景

**牛顿恒等式**（Newton's identities）将幂和 $p_k = \sum a_i^k$ 与初等对称多项式联系起来，是竞赛代数的核心工具。

$$p_k - e_1 p_{k-1} + e_2 p_{k-2} - \cdots = 0$$

其中 $e_1, e_2, \ldots$ 是初等对称多项式（本题中 $e_1=0, e_2=q, e_3=-r$）。

本题通过递推计算幂和，**无需直接展开高次式**，体现了对称函数理论的优雅。等式 $\frac{p_2}{2} \cdot \frac{p_5}{5} = \frac{p_7}{7}$ 中系数 $2, 5, 7$ 恰好满足 $2+5=7$，这并非巧合——背后有更深的组合恒等式结构。
