## 1. 🛠️ 用 Unsloth 和 HF Jobs 免费微调 LLM

## 2. 🗓️ OpenEnv + Calendar Gym：Agent 真实环境评估框架

**来源：** [Hugging Face Blog](https://huggingface.co/blog/openenv-turing)（2026-02-12）  
**领域：** Agent评估 / 工具调用 / 实际部署

Meta 与 Hugging Face 合作开源 OpenEnv，提供标准化框架连接 Agent 与真实系统（非模拟）。Turing 贡献了日历管理 Calendar Gym：任务含访问控制、时态推理、多用户协调，复现真实日历系统约束。评估发现：多步推理是核心瓶颈，任务描述模糊时成功率从 90% 跌至 40%，工具参数错误（而非工具选择错误）是主要失败原因。

**关键词：** `OpenEnv` `Agent评估` `Calendar Gym` `MCP工具调用` `多步推理`

## 3. 📊 Community Evals：去中心化、可复现的模型评测

**来源：** [Hugging Face Blog](https://huggingface.co/blog/community-evals)（2026-02-04）  
**领域：** AI评测 / 基准测试 / 开源生态

MMLU 已饱和（91%+）、GSM8K 达 94%+，但能刷榜的模型在浏览网页、生产级代码等真实任务上仍频繁失败。HF 推出 Community Evals：Dataset 仓库可注册为基准测试、模型在 `.eval_results/` 目录存储评分、社区任意用户可 PR 提交评测结果（附可复现的 eval.yaml）。已上线 MMLU-Pro、GPQA、HLE 等基准，通过 Hub API 开放聚合。

**关键词：** `评测` `Community Evals` `基准测试` `可复现性` `HuggingFace Hub`

## 4. 🤖 NVIDIA Cosmos Reason 2B on Jetson：边缘 VLM 实时推理

**来源：** [Hugging Face Blog](https://huggingface.co/blog/nvidia/cosmos-on-jetson)（2026-02-24）  
**领域：** 边缘AI / VLM部署 / Jetson

NVIDIA 开源 Cosmos Reason 2B（支持思维链视觉推理），本教程展示在 Jetson AGX Thor/Orin/Orin Nano Super 全系列设备上通过 vLLM 完成 FP8 量化部署，配合 Live VLM WebUI 实现摄像头实时推理。Orin 64GB 配置 8192 token 上下文，即使在内存受限的 Orin Nano 上也可通过分块 Prefill 等优化完成部署。

**关键词：** `Cosmos Reason` `Jetson` `VLM` `边缘推理` `vLLM` `FP8`
