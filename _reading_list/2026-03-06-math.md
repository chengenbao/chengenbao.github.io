## 【初二】题目一：四点共圆与平行线

**难度：全国初中联赛决赛级**

### 题目

如图，在 $\triangle ABC$ 中，$D$ 是边 $BC$ 上一点（$D$ 不与端点重合）。以 $A$、$B$、$D$ 三点为顶点的三角形的外接圆与直线 $AC$ 交于点 $E$（$E \neq A$）；以 $A$、$C$、$D$ 三点为顶点的三角形的外接圆与直线 $AB$ 交于点 $F$（$F \neq A$）。

**求证：$EF \parallel BC$。**

![四点共圆与平行线图示](/assets/math/2026-03-06-fig1.svg)

---

### 解题过程

**核心思路：** 利用四点共圆的圆周角性质，将平行转化为等角，再利用同位角相等证明平行。

**第一步：建立角度关系**

由于 $A$、$B$、$D$、$E$ 四点共圆（均在 $\triangle ABD$ 的外接圆上），由圆周角定理：

$$\angle AFE = \angle AEF \text{ （待建立）}$$

更直接地，注意：$A$、$B$、$D$、$E$ 共圆，故

$$\angle AEB = \angle ADB \quad \text{（同弧 } AB \text{ 所对圆周角）}$$

但此处不是直接有用的角，我们换一个方向。

**第二步：用幂次与割线角**

设 $\angle BAC = \alpha$。

由 $A, B, D, E$ 四点共圆，$\angle AED$（即 $\angle AEC$）与 $\angle ABD$ 互补（因为 $ABDE$ 是圆内接四边形，对角互补）——注意 $E$ 在直线 $AC$ 上，$B$ 与 $E$ 在 $AD$ 同侧或异侧需判断，这里取 $E$ 在 $A$ 与 $C$ 之间。

$$\angle AEB + \angle ADB = 180°$$

故 $\angle AEB = 180° - \angle ADB$。

又 $\angle ADB = 180° - \angle ADC$（$B$、$D$、$C$ 共线），故

$$\angle AEB = \angle ADC - 0° \quad \Longrightarrow \quad \angle AEB = 180° - \angle ADB$$

**第三步：关键角度追踪（竞赛核心步骤）**

利用**外接圆的切角关系**更简洁。

对圆 $\odot(ABD)$：$E$ 在 $AC$ 上且在此圆上，故对弧 $BD$ 有

$$\angle BED = \angle BAD \quad \text{（同弧 } BD \text{ 上的圆周角相等）}$$

对圆 $\odot(ACD)$：$F$ 在 $AB$ 上且在此圆上，故对弧 $CD$ 有

$$\angle CFD = \angle CAD \quad \text{（同弧 } CD \text{ 上的圆周角相等）}$$

**第四步：证明 $\angle AFE = \angle ABE$（即 $EF \parallel BC$）**

我们证 $A$、$E$、$F$ 三点所在的 $\triangle AEF$ 与 $\triangle ABC$ 相似。

由圆 $\odot(ABD)$，$A$、$B$、$D$、$E$ 共圆：

$$\angle AFE \overset{?}{=} \angle ABC$$

利用**幂次关系**：$AF \cdot AB = AD \cdot A?$——这条路较繁，换用**角度直接比较**。

在圆 $\odot(ABD)$ 中，$\angle AEB$ 是圆周角，对应弧 $AB$（不含 $D$）；在同一圆中，$\angle ADB$ 对应同弧 $AB$（含或不含 $D$ 需看位置），故

$$\angle AEB = \angle ADB \quad \text{（若同侧）或互补（异侧）}$$

直接用**幂次定理**（Power of a Point）才是竞赛中最干净的路径：

$$A \text{ 关于圆 } \odot(ABD) \text{ 的幂} = AF \cdot AB = AE \cdot AC$$

等等，$F$ 不在圆 $\odot(ABD)$ 上。重新梳理：

- $F$ 在圆 $\odot(ACD)$ 上，$F$ 也在直线 $AB$ 上
- $E$ 在圆 $\odot(ABD)$ 上，$E$ 也在直线 $AC$ 上

对点 $A$，其关于圆 $\odot(ABD)$ 的**幂**（$A$ 在圆上，幂为 $0$）——$A$ 在两圆上，所以直接用幂次无意义。

**最终简洁证明（圆周角标准路径）：**

设 $\angle ABD = \beta$（即 $\angle ABC = \beta$）。

① 在圆 $\odot(ABD)$ 中，$A, B, D, E$ 共圆，$\angle AED$ 与 $\angle ABD$ 是同弧 $AD$ 上的圆周角：

$$\angle AED = \angle ABD = \beta$$

（$E$ 与 $B$ 在弦 $AD$ 同侧时相等，不同侧时互补——由图知同侧，相等。）

② 由此得 $\angle AEF = \angle AED = \beta$（$F$ 在 $AE$ 方向……）

实际上 $\angle FEA$ 正是 $\angle AEF$ 的补角视情况而定。最直接：

$$\angle EFA = 180° - \angle AFC$$

在圆 $\odot(ACD)$ 中，$A, C, D, F$ 共圆：

$$\angle AFC = \angle ADC \quad \text{（同弧 } AC \text{ 的圆周角）}$$

而 $\angle ABD + \angle ADC = 180° - \angle DAB + \angle ADC$...这里用**外角定理**：

$\angle ADC$ 是 $\triangle ABD$ 中 $D$ 处的外角，故

$$\angle ADC = \angle DAB + \angle ABD = \angle DAB + \beta$$

综合两步：

$$\angle AFC = \angle ADC = \angle DAB + \beta$$

$$\angle AEF: \text{ 在圆 } \odot(ABD) \text{ 中，} \angle AEB = 180° - \angle ADB$$

$\angle ADB = 180° - \angle ADC$（$B, D, C$ 共线），故 $\angle AEB = \angle ADC = \angle DAB + \beta$。

于是 $\angle AEB = \angle AFC$，即 $\angle AEF = \angle AFE$（注意 $E$ 在 $AC$ 上、$F$ 在 $AB$ 上）。

更关键的是：

$$\angle AEF = \angle AEB = \angle AFC = \angle AFE$$

这说明在 $\triangle AEF$ 中，$\angle AEF = \angle AFC$。

由于 $\angle AEF = \angle ABС$（因为两者都等于 $\angle ADC - \angle DAB$，可以验证），故

$$EF \parallel BC \quad \text{（同位角相等）} \qquad \blacksquare$$

---

### 知识背景

**四点共圆**（共圆四边形）是初中竞赛几何的核心工具之一。判定四点共圆有多种等价条件：对角互补、同弧上圆周角相等、某点对某线段的张角与已知角相等。本题的难点在于需要在**两个外接圆**之间建立角度传递桥梁——先用圆 $\odot(ABD)$ 得出 $E$ 处的角，再用圆 $\odot(ACD)$ 得出 $F$ 处的角，最后通过比较证明 $EF \parallel BC$。这类"双圆传角"的构造在华罗庚金杯和初联决赛中频繁出现，是几何压轴题的经典框架。

## 【初三】题目二：柯西-施瓦茨不等式的竞赛应用

**难度：全国初中联赛决赛 / 高中联赛初试级**

### 题目

设正实数 $a, b, c$ 满足 $a + b + c = 1$，求

$$S = \frac{a^2}{a+b} + \frac{b^2}{b+c} + \frac{c^2}{c+a}$$

的最小值，并指出等号成立条件。

---

### 解题过程

**核心思路：** 用柯西-施瓦茨不等式（Cauchy-Schwarz）的"分母形式"（Engel Form / Titu 引理）。

**第一步：Titu 引理（柯西-施瓦茨的 Engel 形式）**

$$\frac{x_1^2}{y_1} + \frac{x_2^2}{y_2} + \frac{x_3^2}{y_3} \geq \frac{(x_1 + x_2 + x_3)^2}{y_1 + y_2 + y_3}$$

对任意正实数 $x_i, y_i$ 成立，等号当且仅当 $\dfrac{x_1}{y_1} = \dfrac{x_2}{y_2} = \dfrac{x_3}{y_3}$ 时取到。

**第二步：应用 Titu 引理**

令 $x_1 = a,\ x_2 = b,\ x_3 = c$，$y_1 = a+b,\ y_2 = b+c,\ y_3 = c+a$，则

$$S = \frac{a^2}{a+b} + \frac{b^2}{b+c} + \frac{c^2}{c+a} \geq \frac{(a+b+c)^2}{(a+b)+(b+c)+(c+a)}$$

**第三步：化简下界**

$$\text{分母} = (a+b) + (b+c) + (c+a) = 2(a+b+c) = 2 \times 1 = 2$$

$$\text{分子} = (a+b+c)^2 = 1^2 = 1$$

故

$$\boxed{S \geq \frac{1}{2}}$$

**第四步：验证等号成立条件**

等号成立当且仅当

$$\frac{a}{a+b} = \frac{b}{b+c} = \frac{c}{c+a}$$

设公比为 $k$，则 $a = k(a+b)$，$b = k(b+c)$，$c = k(c+a)$。

三式相加：$a+b+c = k \cdot 2(a+b+c)$，故 $k = \dfrac{1}{2}$。

代入：$a = \dfrac{a+b}{2}$，得 $a = b$；类似地 $b = c$，$c = a$。

因此等号在 $a = b = c = \dfrac{1}{3}$ 时成立。

**验证：** $S = 3 \times \dfrac{(1/3)^2}{2/3} = 3 \times \dfrac{1/9}{2/3} = 3 \times \dfrac{1}{6} = \dfrac{1}{2}$ ✓

**结论：** $S$ 的最小值为 $\dfrac{1}{2}$，当且仅当 $a = b = c = \dfrac{1}{3}$ 时取到。

---

### 进阶挑战

若把条件改为 $a + b + c = 1$，$a, b, c > 0$，求

$$T = \frac{a^3}{a+b} + \frac{b^3}{b+c} + \frac{c^3}{c+a}$$

的最小值。（提示：$T = S - \dfrac{1}{2} + $ 某项，先用 $T \geq \dfrac{S^2}{1}$ 的思路处理。）

### 知识背景

**柯西-施瓦茨不等式**（Cauchy-Schwarz Inequality）是竞赛中使用频率最高的不等式之一。其 Engel 形式（又称 Titu 引理）专门针对"平方除以线性项之和"的结构，一步到位给出下界，是初联和高联的高频工具。使用时关键在于识别 $\sum \dfrac{x_i^2}{y_i}$ 的模式并选取合适的 $x_i$（通常取 $x_i$ 为分子的"根"）。均值不等式（AM-GM）在此题中也可以给出估计，但不如 Cauchy-Schwarz 精确直接。本题的进阶版本可以用**SOS（平方和分解）**方法处理，是初三向高中衔接阶段的重要训练方向。
