## 【四年级】蚱蜢跳莲叶：最少跳多少下？

**难度：** ⭐⭐⭐（AMC 8 Problem 18，中等偏难）

**题目来源：** [2023 AMC 8 Problem 18](https://artofproblemsolving.com/wiki/index.php/2023_AMC_8_Problems/Problem_18)

**题目：**

蚱蜢 Greta 坐在一排很长的莲叶上。她每次可以向**右跳 5 格**，或向**左跳 3 格**。问她最少需要跳多少次，才能到达出发点右边第 **2023** 格的莲叶？

**选项：**
- **(A)** 405
- **(B)** 407
- **(C)** 409
- **(D)** 411 ✓
- **(E)** 413

---

### 解题过程

**关键观察：** 向右跳 5 次 + 向左跳 3 次 = 净移动 $(5 \times a - 3 \times b)$ 格，总跳数为 $a + b$。

**建立方程：**

$$5a - 3b = 2023$$

我们要在满足此方程的所有非负整数解 $(a, b)$ 中，**最小化** $a + b$。

**求最小 b：**

对方程取模 5：$-3b \equiv 2023 \pmod{5}$

计算 $2023 \mod 5 = 3$（因为 $2023 = 5 \times 404 + 3$）

所以 $-3b \equiv 3 \pmod{5}$，即 $3b \equiv -3 \equiv 2 \pmod{5}$

两边乘 2（因为 $3 \times 2 = 6 \equiv 1 \pmod 5$，所以 2 是 3 的逆元）：

$$b \equiv 4 \pmod{5}$$

**最小正整数解：** $b = 4$

代入：$5a = 2023 + 3 \times 4 = 2035$，得 $a = 407$

**验证：**
$$5 \times 407 - 3 \times 4 = 2035 - 12 = 2023 \checkmark$$

**总跳数：** $a + b = 407 + 4 = \mathbf{411}$

**答案：** **(D) 411**

---

### 知识背景

这道题考查的是**线性丢番图方程**（整数方程）的最优解问题，核心是用**模运算**快速找出最小的"反跳"次数 $b$。

**本质思路：** 每次向右5格很高效，但不能整除时，需要一点"修正"——偶尔向左跳3格来调整余数。向左跳的次数越少越好，所以我们要找满足 $5a - 3b = 2023$ 的最小 $b$，再算出 $a$，最后求和 $a+b$。

这类思想在算法竞赛中也很常见，叫做**贪心 + 同余分析**。

## 【五年级】列表插值：极差翻倍

**难度：** ⭐⭐⭐（AMC 8 Problem 20，较难）

**题目来源：** [2023 AMC 8 Problem 20](https://artofproblemsolving.com/wiki/index.php/2023_AMC_8_Problems/Problem_20)

**题目：**

向列表 **3, 3, 8, 11, 28** 中插入两个整数，使得：
1. 极差（最大值 - 最小值）翻倍；
2. 众数（mode）不变；
3. 中位数（median）不变。

问这两个附加数的最大可能之和是多少？

**选项：**
- **(A)** 56
- **(B)** 57
- **(C)** 58
- **(D)** 60 ✓
- **(E)** 61

---

### 解题过程

**分析原列表的统计量：**

原列表（从小到大排列）：3, 3, 8, 11, 28

| 统计量 | 值 |
|--------|-----|
| 极差 | $28 - 3 = 25$ |
| 中位数（第3个数）| $8$ |
| 众数 | $3$（出现2次） |

**极差翻倍后：** 新极差 = $50$，即 $\max - \min = 50$。

**扩大极差的两种方案：**

- **方案A：** 最大值增大，即引入一个数 $= 53$（$3 + 50 = 53$），最小值保持 3
- **方案B：** 最小值减小，即引入 $-22$（$28 - 50 = -22$），最大值保持 28

**方案A 中最大化两数之和：**

插入 $53$ 和另一个数 $x$，共 7 个数，中位数是**第 4 个数** = 8。

排好序后要满足：
- 第 4 个数 $= 8$（中位数不变）
- 3 仍是众数（3 出现最多次）

若 $x \leq 8$（放在 8 的左边）：
$$3, 3, x, 8, 11, 28, 53$$
第 4 个数 $= 8$ ✓

为保持众数为 3（3 出现2次，$x$ 不能等于其他已有的数使其超过2次），$x$ 可以是任意 $\leq 8$ 且不等于 3 的整数，最大取 $x = 7$。

验证 $x = 7$：$3, 3, 7, 8, 11, 28, 53$
- 中位数（第4个）$= 8$ ✓
- 众数 $= 3$（出现2次，最多）✓
- 极差 $= 53 - 3 = 50$ ✓

两数之和 $= 7 + 53 = \mathbf{60}$

**方案B 中最大化两数之和：**

插入 $-22$ 和另一个数 $y$，两数之和最大为 $y - 22$。即使 $y$ 取最大合理值，也难以超过 60（比如 $y = 28$，和仅 $28 + (-22) = 6$）。

**答案：** **(D) 60**（插入 7 和 53）

---

### 知识背景

这道题综合考查**数据统计**中的三个核心概念：

- **极差（Range）：** 最大值与最小值之差，衡量数据的分散程度
- **中位数（Median）：** 排序后中间位置的数，插入新数后位置会改变
- **众数（Mode）：** 出现次数最多的数，新数不能破坏原众数的"第一名"地位

解题关键是同时满足三个约束，用**枚举 + 验证**策略逐步缩小范围，最后找到使两数之和最大的合法方案。
