## 【初二】整除性构造

**题目**

设正整数 $n$ 满足：$n^2 + 3n + 5$ 能被 $n + 1$ 整除。求所有满足条件的正整数 $n$。

---

### 解题过程

**关键构造：利用带余除法消去高次项。**

注意到 $n+1$ 是除式，将被除式用 $n+1$ 表示：

$$n^2 + 3n + 5 = n(n+1) + 2n + 5 = n(n+1) + 2(n+1) + 3$$

因此

$$n^2 + 3n + 5 = (n+1)(n+2) + 3$$

若 $(n+1) \mid (n^2 + 3n + 5)$，则由于 $(n+1) \mid (n+1)(n+2)$，必须有

$$(n+1) \mid 3$$

正整数 $n+1$ 的可能值为 $1, 3$（因为 $n+1 \geq 2$，故 $n+1 \in \{3\}$，即需要在 3 的因子中取大于 1 的值）。

$3$ 的正因子为 $1, 3$。由 $n \geq 1$ 知 $n+1 \geq 2$，所以 $n+1 = 3$，即 $\boxed{n = 2}$。

**验证：** $n=2$ 时，$4 + 6 + 5 = 15 = 3 \times 5$，而 $n+1 = 3 \mid 15$ ✓

$$\boxed{n = 2}$$

---

### 知识背景

**整除性构造**是数论竞赛中的核心技巧。核心思想是：若 $d \mid f(n)$，而 $f(n)$ 含有 $d$ 的倍式，则可以将 $f(n)$ 改写为 $d \cdot g(n) + r(n)$，从而把整除条件转化为 $d \mid r(n)$，其中 $r(n)$ 是更简单的表达式。

这一技巧实质上是多项式带余除法的整数版本，在全国初中数学联赛和 AIME 等竞赛中频繁出现。解题关键不是记公式，而是"主动构造"出被除数与除数之间的代数关系，让余数尽量小从而限制解的范围。

## 【初三】韦达定理与对称多项式

**题目**

设 $x_1, x_2$ 是方程 $x^2 - (k+2)x + (k+1) = 0$ 的两个实根（$k$ 为实数），且满足

$$\frac{x_1}{x_2} + \frac{x_2}{x_1} = \frac{10}{3}$$

求 $k$ 的所有可能值。

---

### 解题过程

**第一步：用韦达定理建立基本量。**

由韦达定理：

$$x_1 + x_2 = k+2, \quad x_1 x_2 = k+1$$

**第二步：将目标式转化为基本对称式。**

$$\frac{x_1}{x_2} + \frac{x_2}{x_1} = \frac{x_1^2 + x_2^2}{x_1 x_2}$$

而

$$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2$$

故目标式变为

$$\frac{(x_1+x_2)^2 - 2x_1x_2}{x_1x_2} = \frac{10}{3}$$

代入韦达定理结果：

$$\frac{(k+2)^2 - 2(k+1)}{k+1} = \frac{10}{3}$$

**第三步：化简方程。**

分子展开：

$$(k+2)^2 - 2(k+1) = k^2 + 4k + 4 - 2k - 2 = k^2 + 2k + 2$$

所以

$$\frac{k^2 + 2k + 2}{k+1} = \frac{10}{3}$$

注意 $k^2 + 2k + 2 = (k+1)^2 + 1$，令 $t = k+1$：

$$\frac{t^2 + 1}{t} = \frac{10}{3}$$

$$3(t^2+1) = 10t$$

$$3t^2 - 10t + 3 = 0$$

$$t = \frac{10 \pm \sqrt{100-36}}{6} = \frac{10 \pm 8}{6}$$

故 $t = 3$ 或 $t = \dfrac{1}{3}$，即 $k = 2$ 或 $k = -\dfrac{2}{3}$。

**第四步：验证两根存在（判别式 $\Delta \geq 0$）且 $x_1 x_2 \neq 0$（避免分母为零）。**

- 要求 $x_1 x_2 = k+1 \neq 0$，即 $k \neq -1$，两值均满足。
- 判别式 $\Delta = (k+2)^2 - 4(k+1) = k^2 \geq 0$，对所有实数 $k$ 成立 ✓

因此

$$\boxed{k = 2 \quad \text{或} \quad k = -\dfrac{2}{3}}$$

---

### 知识背景

**韦达定理（根与系数关系）**将方程的根之和与积直接与系数相连，是代数竞赛的基础工具。更深层的技巧是**对称多项式基本定理**：任何关于 $x_1, x_2$ 的对称多项式都可以用初等对称多项式 $e_1 = x_1+x_2$、$e_2 = x_1 x_2$ 表示。

本题的非平凡之处在于：将目标式 $\frac{x_1}{x_2}+\frac{x_2}{x_1}$ 通分后恰好变成 $\frac{e_1^2 - 2e_2}{e_2}$，再令 $t = e_2 = k+1$ 做换元，将原来的二次分式方程整理成标准二次方程，这是竞赛中"换元降次"与"韦达定理综合应用"的典型范式，在华罗庚金杯预选赛和全国初中联赛中常见。
