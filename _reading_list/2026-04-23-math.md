## 【初二】组合计数：分组配对问题

**难度：** ⭐⭐⭐（AMC 10A Problem 9，中等难度，考察组合计数方法）

**题目来源：** [2024 AMC 10A Problem 9](https://artofproblemsolving.com/wiki/index.php/2024_AMC_10A_Problems/Problem_9)

**题目：**

6名初级生和6名高级生组成3支互不相交的队伍，每支队伍共4人（恰好包含2名初级生和2名高级生）。问共有多少种不同的组合方式？

$$\text{(A) } 720 \quad \text{(B) } 1350 \quad \text{(C) } 2700 \quad \text{(D) } 3280 \quad \text{(E) } 8100$$

---

### 解题过程

**思路：分步计数，分别处理初级生和高级生的配对，再合并。**

**第一步：将6名初级生分成3组，每组2人。**

由于3支队伍是无标签的（队伍之间没有编号顺序），需要用无序分组公式：

$$\frac{6!}{(2!)^3 \cdot 3!} = \frac{720}{8 \times 6} = 15 \text{ 种}$$

**第二步：将6名高级生分成3组，每组2人。**

与初级生相同，也是15种。

**第三步：将初级生对子与高级生对子一一对应，合并成队伍。**

3对初级生与3对高级生之间的匹配方式为 $3! = 6$ 种。

**总计：**

$$15 \times 15 \times 6 = \boxed{1350}$$

答案为 **(B)**。

---

### 常见错误

- 把3支队伍当成有编号的有序队伍，导致结果多算了 $3! = 6$ 倍。
- 直接对12个人进行排列，没有利用"每队2初2高"的限制条件。
- 计算无序分组时忘记除以 $(2!)^3$ 或 $3!$。

---

### 知识背景

本题考查的是**无标签分组（无序分组）**的计数方法。当 $n$ 个人被分成 $k$ 组，每组 $m$ 人，且各组之间没有区别时，计算公式为：

$$\frac{n!}{(m!)^k \cdot k!}$$

分母中 $(m!)^k$ 消除了每组内部排列的重复，$k!$ 消除了各组之间顺序的重复。这是计数竞赛中的基本模型，也是学习"二项式系数"和"斯特灵数"的基础。

## 【初三】坐标变换：交换律的几何意义

**难度：** ⭐⭐⭐⭐（AMC 10A Problem 13，需要对坐标变换的代数性质有深入理解）

**题目来源：** [2024 AMC 10A Problem 13](https://artofproblemsolving.com/wiki/index.php/2024_AMC_10A_Problems/Problem_13)

**题目：**

如果两个变换先后顺序不同但结果相同，则称这两个变换**可交换**。考虑坐标平面上如下4种变换：

1. 向右平移2个单位
2. 绕原点逆时针旋转90°
3. 关于 $x$ 轴做反射
4. 以原点为中心，放大系数为2的伸缩变换

在这4种变换的6个不同对子中，有多少对是可交换的？

$$\text{(A) } 1 \quad \text{(B) } 2 \quad \text{(C) } 3 \quad \text{(D) } 4 \quad \text{(E) } 5$$

---

### 解题过程

设4种变换分别为：
- $T$：平移，$(x, y) \mapsto (x+2, y)$
- $R$：旋转90°，$(x, y) \mapsto (-y, x)$
- $F$：关于 $x$ 轴反射，$(x, y) \mapsto (x, -y)$
- $D$：伸缩，$(x, y) \mapsto (2x, 2y)$

**验证6对：**

**① $T$ 与 $F$（平移 & 反射）：** 可交换 ✓

$$F(T(x,y)) = F(x+2, y) = (x+2, -y)$$
$$T(F(x,y)) = T(x, -y) = (x+2, -y)$$

**② $R$ 与 $D$（旋转 & 伸缩）：** 可交换 ✓

$$D(R(x,y)) = D(-y, x) = (-2y, 2x)$$
$$R(D(x,y)) = R(2x, 2y) = (-2y, 2x)$$

**③ $F$ 与 $D$（反射 & 伸缩）：** 可交换 ✓

$$D(F(x,y)) = D(x,-y) = (2x, -2y)$$
$$F(D(x,y)) = F(2x, 2y) = (2x, -2y)$$

**④ $T$ 与 $R$（平移 & 旋转）：** 不可交换 ✗

$$R(T(x,y)) = R(x+2, y) = (-y, x+2)$$
$$T(R(x,y)) = T(-y, x) = (-y+2, x)$$

**⑤ $T$ 与 $D$（平移 & 伸缩）：** 不可交换 ✗

$$D(T(x,y)) = D(x+2, y) = (2x+4, 2y)$$
$$T(D(x,y)) = T(2x, 2y) = (2x+2, 2y)$$

**⑥ $R$ 与 $F$（旋转 & 反射）：** 不可交换 ✗

$$F(R(x,y)) = F(-y, x) = (-y, -x)$$
$$R(F(x,y)) = R(x, -y) = (y, x)$$

**共有3对可交换**，答案为 **(C)**。

---

### 知识背景

本题考查**坐标变换的代数性质**。在线性代数中，变换的合成对应矩阵的乘法。矩阵乘法通常不满足交换律，但有些特殊变换对之间是可交换的：

- 以同一原点为中心的**均匀伸缩**与旋转、反射均可交换（因为均匀伸缩等价于数乘单位矩阵的倍数）。
- **平移**不是线性变换（不保持原点），与旋转/伸缩通常不可交换。
- 平移与关于 $x$ 轴的反射可交换，原因在于平移方向（$x$ 方向）与反射轴（$x$ 轴）平行，两者在 $y$ 坐标上互不干扰。

这是理解仿射变换、对称群的重要基础。
