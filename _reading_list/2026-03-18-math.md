## 【初二】最优赛制中的概率方程

**难度：** ⭐⭐⭐⭐（2024 AMC 10A Problem 17，联赛选拔级）

**题目来源：** [2024 AMC 10A Problem 17](https://artofproblemsolving.com/wiki/index.php/2024_AMC_10A_Problems/Problem_17)

**题目：**

甲、乙两队进行"三局两胜制"比赛：最多打 3 局，先赢 2 局者获胜。第一局在甲队主场进行，其余局在乙队主场进行。甲队在主场赢球的概率为 $$\frac{2}{3}$$，在客场赢球的概率为 $$p$$，每局结果相互独立。已知甲队最终赢得系列赛的概率恰好为 $$\frac{1}{2}$$，则 $$p = \frac{1}{2}(m - \sqrt{n})$$，求 $$m + n$$。

---

### 解题过程

甲队赢得系列赛的所有情况（A 表示甲赢，B 表示乙赢）：

- **AA**：甲赢第1、2局。概率 = $$\frac{2}{3} \cdot p$$
- **ABA**：甲赢1、3局，乙赢第2局。概率 = $$\frac{2}{3}(1-p) \cdot p$$
- **BAA**：乙赢第1局，甲赢2、3局。概率 = $$\frac{1}{3} \cdot p^2$$

令三种情况之和等于 $$\frac{1}{2}$$：

$$\frac{2}{3}p + \frac{2}{3}(1-p)p + \frac{1}{3}p^2 = \frac{1}{2}$$

两边乘以 6：

$$4p + 4p(1-p) + 2p^2 = 3$$

$$4p + 4p - 4p^2 + 2p^2 = 3$$

$$-2p^2 + 8p - 3 = 0 \Rightarrow 2p^2 - 8p + 3 = 0$$

用求根公式：$$p = \frac{8 \pm \sqrt{64 - 24}}{4} = \frac{8 \pm \sqrt{40}}{4} = \frac{4 \pm \sqrt{10}}{2}$$

由于 $$p \in [0,1]$$，取减号：$$p = \frac{4 - \sqrt{10}}{2} = \frac{1}{2}(4 - \sqrt{10})$$

故 $$m = 4,\ n = 10$$，答案为 $$m + n = \boxed{14}$$。

### 知识背景

本题考查**条件概率**与**分类枚举**。核心技巧是对竞赛结果按"甲赢"的所有路径穷举（AA / ABA / BAA），分别写出概率表达式后相加建立方程。解方程用到一元二次公式，并利用概率的值域 $$[0,1]$$ 排除增根。这类"比赛系列赛概率"题在初中竞赛中频繁出现。

## 【初三】两矩形同圆——圆幂定理求线段

**难度：** ⭐⭐⭐⭐⭐（2024 AIME I Problem 5，全美邀请赛级）

**题目来源：** [2024 AIME I Problem 5](https://artofproblemsolving.com/wiki/index.php/2024_AIME_I_Problems/Problem_5)

**题目：**

矩形 $$ABCD$$ 和矩形 $$EFGH$$ 满足：$$D, E, C, F$$ 四点共线；$$A, D, H, G$$ 四点共圆。已知 $$BC = 16$$，$$AB = 107$$，$$FG = 17$$，$$EF = 184$$，求线段 $$CE$$ 的长。

![几何图示](/assets/math/2026-03-18-fig1.svg)

---

### 解题过程（圆幂定理法）

设 $$DE = x$$（即 $$CE = AB - DE - ... $$，稍后算出）。

**关键构造：** 延长 $$AD$$ 和 $$GH$$ 直至两线交于点 $$P$$。

由矩形性质：
- $$AD = BC = 16$$，$$GH = EF - 	ext{(待定)} = 17$$（即 $$FG = 17$$，故 $$GH$$ 长同矩形宽）
- $$PD = $ $ （上方矩形宽度）$$ = FG = 17$$
- $$PA = PD + DA = 17 + 16 = 33$$
- $$PH = DE = x$$（$$P$$、$$H$$ 在同一水平线，水平距离即 $$DE$$）
- $$PG = PH + HG = x + EF = x + 184$$

因为 $$A, D, H, G$$ 四点共圆，点 $$P$$ 在圆外，对割线 $$PAD$$ 和割线 $$PHG$$ 用**圆幂定理**：

$$PD \cdot PA = PH \cdot PG$$

$$17 \times 33 = x(x + 184)$$

$$561 = x^2 + 184x$$

$$x^2 + 184x - 561 = 0$$

分解因式：$$(x - 3)(x + 187) = 0$$

由于 $$x > 0$$，故 $$x = DE = 3$$。

则 $$CE = AB - BC - DE \cdot ... $$ 

实际上由共线条件：$$CE = AB - (DE) \cdot$$ ——注意 $$C$$ 在 $$E$$ 右侧（看图），所以

$$CE = AB - DE = 107 - 3 = 104$$

答案：$$CE = \boxed{104}$$。

### 知识背景

本题综合运用**圆幂定理（Power of a Point）**：若点 $$P$$ 在圆外，两条过 $$P$$ 的割线分别交圆于 $$A, B$$ 和 $$C, D$$，则 $$PA \cdot PB = PC \cdot PD$$。这是初三竞赛的核心定理之一，常与矩形、平行线等结合出现，难点在于识别"共圆四点"并构造合适的割线位置。
