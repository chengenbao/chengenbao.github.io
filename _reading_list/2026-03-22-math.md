## 【四年级】等间距数列求和

**难度：** ⭐⭐⭐（AMC 8 Problem 14，等差数列基础）

**题目来源：** [2026 AMC 8 Problem 14](https://ivyleaguecenter.org/wp-content/uploads/2025/11/2026-amc-8-problems-and-answers.pdf)

**题目：**

Jami 在数轴上选了三个**等间距**的整数。第一个数与第二个数的和为 $40$，第二个数与第三个数的和为 $60$。这三个数的总和是多少？

- (A) $70$
- (B) $75$
- (C) $80$
- (D) $85$
- (E) $90$

**正确答案：(B) $75$**

---

### 解题过程

**方法一：设未知数（代数法）**

设三个等间距整数为 $a-d$、$a$、$a+d$（$a$ 为中间数，$d$ 为公差）。

由题意列方程：

$$\text{条件1：} (a - d) + a = 40 \implies 2a - d = 40$$

$$\text{条件2：} a + (a + d) = 60 \implies 2a + d = 60$$

两式相加消去 $d$：

$$4a = 100 \implies a = 25$$

代回求 $d$：$2(25) - d = 40 \implies d = 10$

三个数为：$15$、$25$、$35$

$$15 + 25 + 35 = \boxed{75}$$

**方法二：直接观察（巧妙法）**

两个条件的和相加：

$$[(a_1 + a_2) + (a_2 + a_3)] = 40 + 60 = 100$$

即 $a_1 + 2a_2 + a_3 = 100$。

由等差性质，$a_1 + a_3 = 2a_2$，代入得：

$$4a_2 = 100 \implies a_2 = 25$$

又 $a_1 + a_2 = 40$，所以 $a_1 = 15$；$a_3 = 35$。

$$S = a_1 + a_2 + a_3 = 15 + 25 + 35 = \boxed{75}$$

---

### 知识背景

这道题考查**等差数列（Arithmetic Sequence）**的基本性质。

等差数列的关键性质：
1. **连续三项之和** = $3 \times$（中间项）
2. **首末两项之和** = $2 \times$（中间项）
3. 中间项是首末项的**算术平均数**

本题还涉及**二元一次方程组**的解法：两式相加/相减消去一个未知量，是初等代数的基本技巧。

等差数列在小学奥数中非常常见，通项公式 $a_n = a_1 + (n-1)d$ 是必须掌握的知识。

## 【六年级】追及问题——人与狗

**难度：** ⭐⭐⭐⭐（AMC 8 Problem 19，行程问题综合）

**题目来源：** [2026 AMC 8 Problem 19](https://ivyleaguecenter.org/wp-content/uploads/2025/11/2026-amc-8-problems-and-answers.pdf)

**题目：**

Miguel 带着狗 Luna 散步。到达公园入口时，Miguel 将球扔向前方（球滚动后停在一棵树旁），随后继续以匀速步行。Luna 冲向球，到达球后立刻衔球返回 Miguel。**Luna 的速度是 Miguel 步行速度的 5 倍。**

当 Luna 把球带回 Miguel 时，Miguel 走了从入口到树这段距离的几分之几？

- (A) $\dfrac{1}{6}$
- (B) $\dfrac{1}{5}$
- (C) $\dfrac{1}{4}$
- **(D) $\dfrac{1}{3}$**
- (E) $\dfrac{2}{5}$

**正确答案：(D) $\dfrac{1}{3}$**

---

### 解题过程

**建立模型：**

设 Miguel 速度为 $v$，则 Luna 速度为 $5v$。

设入口到树的距离为 $D$，全程总时间为 $t$（Luna 去程 + 回程）。

**关键思路：** Miguel 走的总距离为 $x = vt$，而 Luna 的路径是：先向前跑到树（距离 $D$），再返回追上 Miguel（距离 $D - x$）。

Luna 的总路程：

$$5v \cdot t = D + (D - x) = 2D - x$$

Miguel 的总路程：

$$v \cdot t = x$$

因此 $t = \dfrac{x}{v}$，代入 Luna 的方程：

$$5v \cdot \frac{x}{v} = 2D - x$$

$$5x = 2D - x$$

$$6x = 2D$$

$$x = \frac{D}{3}$$

**答案：** Miguel 走了 $\dfrac{D}{3}$，即入口到树距离的 $\boxed{\dfrac{1}{3}}$。

---

### 验证

设 $D = 6$（单位），则 $x = 2$，$v = 1$，$5v = 5$，总时间 $t = 2$。

- Luna 去程：跑了 $6$ 的距离，用时 $\dfrac{6}{5} = 1.2$ 秒
- 此时 Miguel 在位置 $1.2$，Luna 在树（位置 $6$），两者相距 $4.8$
- Luna 回程：两者相向而行，速度之和 $= 1 + 5 = 6$，用时 $\dfrac{4.8}{6} = 0.8$ 秒
- 总时间：$1.2 + 0.8 = 2$ 秒 ✅
- Miguel 总路程：$1 \times 2 = 2 = \dfrac{6}{3}$ ✅

---

### 知识背景

这是经典的**追及/相遇问题**，核心公式：

$$\text{相遇时间} = \frac{\text{初始距离}}{\text{速度之和}}$$

$$\text{追及时间} = \frac{\text{初始距离}}{\text{速度之差}}$$

解题关键是用**方程法**：设 Miguel 走的总路程为 $x$，然后通过"时间相等"建立方程。类似问题（如"火车相遇"、"游泳池折返"）都遵循同样的分析框架：

1. 确定每个对象的速度
2. 用总时间建立等式
3. 注意 Luna 回程时 Miguel 已经移动——两者路程之和等于当时的间距

这类题目在初中竞赛中频繁出现，方程法是最稳妥的解法。
